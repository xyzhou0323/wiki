# PAQ 珀斯述情障碍问卷

**PAQ**（Perth Alexithymia Questionnaire）是一份评估**述情障碍**（Alexithymia）的自填工具，由 Preece 等人于2018年开发。

> **本站测试**：[PAQ 珀斯述情障碍问卷](https://neuroxyz.cn/tests/paq.html)
> **中文版提示**：本测试目前仅有英文原版常模，**尚无中文版常模**。本站提供的百分位、阈值等解读数据均基于英文原版，缺少本土化验证。中文语境下的分数解读存在文化差异风险，结果**仅供参考**，不构成任何临床判断依据。

## 什么是述情障碍

**述情障碍**指识别和描述自身情绪状态的困难。它并非[[孤独谱系障碍]]的专属特征——在一般人群中也有分布——但在孤独谱系群体中发生率显著较高（约40-65%）。述情障碍与许多心理健康困难（尤其是焦虑和抑郁）密切相关。

## 测试结构

- **24题**，约8分钟完成
- 核心区分：**正性情绪 vs. 负性情绪**——述情障碍在这两种情绪上的困难可能不对称
- 三个子维度：情绪识别困难（DIF）、情绪描述困难（DDF）、外向性思维（EOT）
- 每题采用 **7点 Likert 量表**：

| 分值 | 标签 |
|------|------|
| 1 | 非常不同意 |
| 2 | 不同意 |
| 3 | 有些不同意 |
| 4 | 中立 |
| 5 | 有些同意 |
| 6 | 同意 |
| 7 | 非常同意 |

### 题目-分量表对应

| 分量表 | 代码 | 题目（1-indexed） | 含义 |
|--------|------|-------------------|------|
| 负性-情绪识别困难 | N-DIF | 2, 8, 14, 20 | 难以识别自己的负性情绪 |
| 正性-情绪识别困难 | P-DIF | 5, 11, 17, 23 | 难以识别自己的正性情绪 |
| 负性-情绪描述困难 | N-DDF | 1, 7, 13, 19 | 难以向他人描述自己的负性情绪 |
| 正性-情绪描述困难 | P-DDF | 4, 10, 16, 22 | 难以向他人描述自己的正性情绪 |
| 总体外向性思维 | G-EOT | 3, 6, 9, 12, 15, 18, 21, 24 | 较少关注内在情绪，偏向外部导向思维 |

### 复合分数公式

| 复合分      | 代码    | 计算公式                                              |
| -------- | ----- | ------------------------------------------------- |
| 总体情绪识别困难 | G-DIF | = N-DIF + P-DIF                                   |
| 总体情绪描述困难 | G-DDF | = N-DDF + P-DDF                                   |
| 负性情绪评估困难 | N-DAF | = N-DIF + N-DDF                                   |
| 正性情绪评估困难 | P-DAF | = P-DIF + P-DDF                                   |
| 总体情绪评估困难 | G-DAF | = G-DIF + G-DDF (= N-DIF + P-DIF + N-DDF + P-DDF) |
| 述情障碍总分   | TOTAL | = 全部 24 题得分之和                                     |

## 结果解读

### 整体分级

基于总分的分级（参照常模均值 82，SD=31）：

| 总分 | 分级 |
|------|------|
| < 51 | 无述情障碍（低于均值-1SD） |
| 51 - 112 | 平均水平（均值±1SD 范围内） |
| >= 113 | 述情障碍（高于均值+1SD） |

> **注意**：述情障碍可能是一个连续谱系（低 -> 中 -> 高），而非截然不同的分类。以上分级仅为便于理解的参考框架。

### 群体特征提示

- 普通人群中，**负性情绪维度**（N-DIF、N-DDF）通常得分**高于**正性情绪维度（P-DIF、P-DDF）
- 述情障碍人群的负性情绪困难通常更为突出

## 在神经多样性范式中的位置

述情障碍是[[神经多样性范式]]与[[常态范式]]之间紧张关系的典型案例：

- 述情障碍确实造成个体的主观困扰——消除它的努力有合理的基础
- 但分类须谨慎：情绪觉察方面的差异**不必然**是述情障碍——某些差异反映的可能是不同的情绪加工风格，而非能力缺陷
- [[神经多样性范式]]要求我们区分"造成困扰"和"仅偏离规范"——前者可以干预，后者不应被病理化

## 与其他概念的关系

述情障碍与内感觉密切相关——对身体信号的觉察困难可能导致情绪觉察困难（[[MAIA-2]]评估内感觉）。述情障碍也与同理心有关——识别自身情绪的困难可能延伸到理解他人情绪（[[PES]]评估同理心）。
