# RBQ-3 重复行为

**RBQ-3**（Repetitive Behaviour Questionnaire 3）是一份评估重复行为模式的工具，由 Jones 等人于2024年开发。覆盖从15月龄婴幼儿到成人的全年龄段，可自评也可由他人评估。

> **本站测试**：[RBQ-3 重复行为](https://neuroxyz.cn/tests/rbq3.html)
> **中文版提示**：本测试目前仅有英文原版常模，**尚无中文版常模**。本站提供的百分位、阈值等解读数据均基于英文原版，缺少本土化验证。中文语境下的分数解读存在文化差异风险，结果**仅供参考**，不构成任何临床判断依据。

## 测试结构

- **20题**，约5分钟完成
- **自评适用**：13 岁以上无智力障碍个体；他评适用于 15 月龄以上全年龄段
- **计分规则**（关键）：选项索引需转换为得分（并非简单的 1/2/3/4）
  - 4 点量表题（Q1-Q19）：选项映射为 **[1, 2, 3, 3]**（即第2和第3个选项同为 3 分）
  - 3 点量表题（Q20）：选项映射为 **[1, 2, 3]**
- **两类量表**：
  - **Q1-Q6 频率量表**：以每日行为次数度量（如"从不或很少"到"每天30次或更多"）
  - **Q7-Q20 严重程度量表**：以行为强度/影响程度度量（如"从不或很少"到"严重或频繁"）
- 两个子维度：
  - **重复感觉运动行为**（RSMB）：如摆动身体、旋转物体等
  - **坚持同一性**（IS）：对日常程序和一致性的强烈需求
- **不参与维度计分的题目**（维度标记为"NONE"）：Q1, Q11, Q20
  - 这三题仅计入总分和总均分，不计入 RSMB 或 IS 子维度均分

## 结果解读

### 均分分级体系

RBQ-3 使用各维度均分（而非粗分）进行分级，四档评估：

| 等级 | 总均分 | RSMB 均分 | IS 均分 |
|------|--------|-----------|---------|
| 从不或很少 | < 1.75 | < 1.75 | < 1.78 |
| 轻微或偶尔 | 1.75 - 2.49 | 1.75 - 2.49 | 1.78 - 2.55 |
| 明显或经常 | 2.50 - 3.24 | 2.50 - 3.24 | 2.56 - 3.32 |
| 严重或频繁 | ≥ 3.25 | ≥ 3.25 | ≥ 3.33 |

### 常模对比

| 人群 | 总均分 | RSMB 均分 | IS 均分 |
|------|--------|-----------|---------|
| 典型人群 | ~1.35 | ~1.29 | ~1.36 |
| 孤独谱系群体 | ~2.48 | ~2.25 | ~2.28 |

### 分级解读

各维度均分对应的特征解读：

#### 总均分

| 均分 | 解读 |
|------|------|
| 从不或很少（< 1.75） | 行为模式非常灵活，很少表现出重复行为特征，日常变化不会造成困扰 |
| 轻微或偶尔（1.75 - 2.49） | 偶尔表现出一些重复性行为特征，但整体灵活度良好 |
| 明显或经常（2.50 - 3.24） | 经常表现出重复行为特征，可能对日常生活造成一些困扰 |
| 严重或频繁（≥ 3.25） | 表现出强烈的重复行为特征并极度偏好稳定与一致性，可能显著影响日常生活，对变化容易感到焦虑 |

#### RSMB（重复感觉运动行为）

| 均分 | 解读 |
|------|------|
| 从不或很少（< 1.75） | 少有重复的感觉运动行为，身体动作模式灵活多样 |
| 轻微或偶尔（1.75 - 2.49） | 偶尔表现出重复性感觉运动行为，但整体身体动作较灵活 |
| 明显或经常（2.50 - 3.24） | 经常表现出重复性感觉运动行为，可能是感官自我调节策略或习惯性行为模式 |
| 严重或频繁（≥ 3.25） | 频繁表现出重复性感觉运动行为，可能对日常活动产生明显影响 |

#### IS（坚持同一性）

| 均分 | 解读 |
|------|------|
| 从不或很少（< 1.78） | 对日常变化的接受度很高，适应新环境的能力强 |
| 轻微或偶尔（1.78 - 2.55） | 对环境稳定性有一定需求，但通常能够适应日常变化 |
| 明显或经常（2.56 - 3.32） | 较为依赖环境的稳定性和日常的可预测性，改变惯例可能引起不适或焦虑 |
| 严重或频繁（≥ 3.33） | 高度依赖环境的稳定性和可预测性，改变惯例通常引起明显不适或焦虑 |

原始总分范围为 20-79 分。RSMB 子量表原始分范围 8-32 分，IS 子量表原始分范围 9-36 分。结果以雷达图可视化三个维度的均分剖面。

## 在神经多样性范式中的位置

重复行为在传统[[常态范式]]下被定义为"刻板行为障碍"。在[[神经多样性范式]]下，RBQ-3 评估的这些行为被重新理解为**自我调节**方式——包括感官调节、情绪调节和认知调节。坚持同一性也被理解为对可预测性的合理需求，而非僵化。

## 与其他版本的关系

RBQ-3 是对[[RBQ-2A]]（成人版）的更新和扩展，统一了全年龄段的评估标准。三个版本的演进反映了对重复行为理解的深化：从单一维度到多维结构。
