# 阿斯至上主义（Aspie Supremacy）

**术语提出者**：Mel Baggs（已故多重残障倡导者，2010年）
**核心定义**：认为"阿斯"拥有非凡能力，因此他们不仅有价值，而且比其他人更优越。

## 本质

阿斯至上主义的核心是建立一种**内部等级制度**——通过强调自身的"优越性"来争取"门票"（社会认可），同时与更广泛的残障群体划清界限。这包括其他孤独谱系者，也包括其他非孤独谱系残障群体。

## 历史源流

从优生学到社会达尔文主义，阿斯至上主义的根源可追溯到 Asperger 本人——"这些小教授将成为帝国优秀的密码破译者"。区分"有社会用途"和"无用途"的残障儿童，是纳粹时期优生学逻辑的延续。

### 首篇系统学术分析：de Hooge (2019)

[[deHooge2019|de Hooge (2019)]] 发表了阿斯至上主义的首篇系统性学术分析，采用情境知识认识论（standpoint epistemology），以 #ActuallyAutistic 身份书写。论文的核心理论贡献包括：

**两个面向，同一硬币**：阿斯至上主义由（1）反孤独谱系的健全中心主义和（2）阿斯主体的优越性建构共同构成——两者不是对立而是互补：前者将孤独谱系者去人化，后者将特定类型的孤独谱系者（白人、男性、有经济价值）推入人类规范性的中心。

**与 Lyotard 后现代主义的平行**：de Hooge 借用 Richard J. Bernstein 的"宏大而诱人的非此即彼"——"要么存在知识的固定基础（不受不请自来的 stimming 威胁），要么我们无法逃脱道德相对主义（表现为对阿斯偏执的纵容）"——指出道德相对主义被用作维护客观主义规范性的手段。正如 Lyotard 式后现代主义表面解构一切元叙事、实则保留白人中心参照点，阿斯至上主义表面去中心化人类、实则重新中心化白人男性气质。

**后人类主义的局限**：阿斯至上主义表明，"人类规范性不会因为口头支持后人类主义而被消除"——后人类姿态可以轻易被收编来重新确认而非颠覆既有的权力等级。

**极端男性脑（EMB）理论的批判**：Baron-Cohen 的 EMB 理论将孤独谱系建构为"男性智能的极端变体"，其实际应用仅覆盖被视为高功能的群体，掩盖了诊断中的种族和性别差异——非白人/非男性孤独谱系者更常被诊断为伴随智力障碍。

**酷残 vs. 阿斯至上**：阿斯至上主义者拒绝酷残策略——后者如 Mel Baggs 在《In My Language》中所展示的，通过与非人类现象的内在互动（intra-action）颠覆象征界；前者则试图通过挤入人类规范性中心来获取特权。

## 表现

### 功利主义论证
- Heilker：孤独谱系是"对普通神经系统的显著改进"（a marked improvement over normal neurology）
- "这些高功能天才可以转化成非凡的生产力，所以他们有价值"
- "阿斯才是人类进化的下一个高级阶段"

### 特定媒体形象
- 福尔摩斯、谢尔顿（《生活大爆炸》）、唐·蒂尔曼（《The Rosie Project》）
- 高度聚焦于"高智商"技术男性的刻板印象

## 阿斯至上主义 ≠ 神经多样性范式

| 阿斯至上主义             | 神经多样性范式               |
| ------------------ | --------------------- |
| 划分等级：基于认知能力和社会价值   | 拒绝排序：承认多样性的固有价值       |
| 排他性：排斥智力障碍等群体      | 包容性：关注交叉性，寻求更广泛的解放    |
| 寻求门票：追求精英阶层地位与社会承认 | 拆除常态帝国：寻求集体解放，建设无障碍环境 |
| 个人竞争自豪："我比他们更有用"   | 本质尊严：认可人的内在价值         |

## 可能带来的风险

1. **表面化接纳**：系统吸纳少数精英人士，但更加拒绝结构性变革
2. **分裂与对立**：利用内部差异来否定其他成员的代表权，抹除合作的可能性
3. **极右翼和武器化**：基于"被排斥者的虚假优越感"受到极端组织利用
4. **优生学合流**：区分"值得保留"的人和"应该去除/预防"的人
5. **社会公正风险**：固化特权、抹除交叉性，最需要支持的人被视为"不值得关注"
6. **心理健康风险**："高功能"身份的脆弱性，为抱有优越感的个体同样带来伪装、耗竭和自杀风险

## 重要区分

展示自身优势（从擅长的事中获得快乐、建立自尊）并不等同于阿斯至上主义。
关键在于：**能力不决定人的价值和权利**。

正如 Chapman 和 Fletcher-Watson 在[[ChapmanFletcherWatson2026|Chapman & Fletcher-Watson (2026)]]中所言：

> 每个神经殊异的个体都有充分的权利展示自己的才能，并反驳那些认为他们缺乏能力的假设，但我们必须共同注意，在这样做时，不应将才能或智力作为衡量个人价值的替代标准。
