# Arnold et al. (2026) — 掩饰的概念化与方法论问题

引用：Arnold, W. M., Bitsika, V., & Sharpley, C. F. (2026). Camouflaging and autism: Conceptualisation and methodological issues. *Autism, 30*(5), 1131–1146. [https://doi.org/10.1177/13623613261420085](https://doi.org/10.1177/13623613261420085)

**作者**：Wayne M Arnold, Vicki Bitsika, Christopher F Sharpley（University of New England, Australia）
**发表**：2026年，*Autism* 期刊（SAGE）
**性质**：Review（综述）
**立场声明**：第一作者为孤独谱系人士，常在日常生活中进行掩饰

## 核心问题

对 389 篇掩饰研究（2000–2025）进行批判性综述，揭示该领域在**概念化**和**方法论**两个层面的系统性缺陷——"一个领域在用不同的词研究不同的事"。

## 概念化问题

### 1. 术语混乱

研究发现 **4 类概念不一致**：

| 问题类型 | 具体表现 |
|----------|---------|
| 用词不统一 | 已确立术语 26 个（impression management、concealment...）+ 新创术语 14 个（adaptive morphing、social morphing...） |
| 引用不一致 | 有些研究引用已确立的概念文献（印象管理、自我呈现理论），多数不引用 |
| 使用不一致 | 59.1% 多词互换使用；19.0% 多词区分使用；19.0% 单词使用；2.8% 同时混用 |
| 定义不一致 | 有些研究不给定义，有些给出模糊/循环定义 |

尽管 CAT-Q 是主导测量工具，仅 **15.9%** 的研究采用了与 CAT-Q 一致的三维概念框架（伪装+补偿+融入）。

### 2. 与已有概念的混淆

论文指出掩饰行为与以下非孤独谱系特有的心理学概念高度重叠，但研究者在发展 CAT-Q 时未加区分：

- **印象管理/自我呈现**（Impression Management/Self-Presentation）：Goffman 1959；Schlenker 2012
- **污名隐藏**（Concealment）：Pachankis 2007 污名理论
- **社会学习**（Social Learning）：Bandura 1977, 1986 — CAT-Q 的"补偿"维度实质可能是正常的社交技能习得
- **自我监控**（Self-Monitoring）：Snyder 1974 — CAT-Q "伪装"维度的大部分题项可能测量的是自我监控
- **社交焦虑**：CAT-Q 总分与社交焦虑的相关经常强于与孤独谱系特征的相关

### 3. "补偿"的残疾主义批判

Marocchini (2023) 指出，"补偿"一词带有**残疾主义**色彩——将孤独谱系人士展示的社交能力归类为"非典型"行为，而非将其视为正常的社交技能习得过程。

## 方法论问题

### 4. CAT-Q 的效度争议

| 支持 | 反对 |
|------|------|
| 信度（内部一致性、重测）良好到优秀 | 与差异法测量弱相关（收敛效度不足） |
| 短版（CATQ-SF）与原版高相关 | 荷兰、日本、瑞典、繁体中文版**未能复现三因子结构** |
| 美国大样本复现三因子结构 | 荷兰和法语版未通过**测量不变性**检验 |
| | "融入"维度与社交焦虑显著重叠 |
| | "伪装"维度未覆盖 RRB（受限重复行为）的掩饰 |

CAT-Q 伪装维度的关键发现：
- **广泛认为该维度与孤独谱系特征无关**
- 区分孤独谱系 vs 非孤独谱系的效应量很小
- 非孤独谱系青少年在两项研究中得分反而**更高**

### 5. 差异法的效度问题

差异法（自我报告 - 观察者报告）的假设是：自我报告的孤独谱系特征高于外部观察时即代表掩饰。但：
- 这种差异可能只是**测量误差**
- 可能反映**自我呈现/社会赞许**而非掩饰本身
- 研究普遍忽略了 De Los Reyes 等建立的方法论文献（差异 ≠ 测量误差的方法论标准）

### 6. 样本推广性不足

| 实际情况 | 研究样本 |
|----------|---------|
| 孤独谱系男女比 3–4:1 | 以成年诊断女性为主体 |
| 全球平均诊断年龄 60 个月 | 绝大多数成年诊断者 |
| 33% 共现智力障碍 | 常规排除智力障碍 |
| 大量高支持需求个体 | 排除语言困难和高支持需求者 |

掩饰行为是否能推广至整个孤独谱系，**尚未建立**。

## 未来方向

1. **概念分析**：进一步区分掩饰 vs. 自我呈现、隐藏、社交焦虑、自我监控、社会学习——这些区分应在开发测量工具**之前**完成
2. **测量改进**：开发分别测量自我呈现、隐藏、社会学习的量表；覆盖非社交孤独谱系特征（刺激自我调控等）的掩饰
3. **样本多样化**：纳入男性、儿童、共现智力障碍和更高支持需求群体
4. **方法规范**：差异法研究采用 De Los Reyes 的方法论标准
5. **临床谨慎**：不将 CAT-Q 作为筛查工具使用，考虑竞争解释（社交焦虑、自我呈现等）
