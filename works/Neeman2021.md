# Ne'eman (2021) — 当残障由行为定义时，结果测量不应鼓励"扮装"

引用：Ne'eman, A. (2021). When disability is defined by behavior, outcome measures should not promote "passing". AMA Journal of Ethics, 23(7), E569-E575. https://doi.org/10.1001/amajethics.2021.569

**作者**：[[Ari-Ne'eman|Ari Ne'eman]]（哈佛大学健康政策博士候选人）
**发表**：*AMA Journal of Ethics* (2021), 23(7), E569-E575
**类型**：伦理学分析

## 核心论证

当残障由行为定义时，临床工作者和研究者面临如何评估临床进展的难题。一些人选择将诊断特质的减少或消除作为成功标准——这暗含将典型外观作为服务的终极目标。Ne'eman 以孤独谱系和神经多样性运动为案例，系统论证了这一做法的伦理问题，并提出了替代性的评估框架。

## 测量工具并非中立

- 许多残障通过生物标志物诊断，但另一些只能通过行为识别——在后者中，临床工作者往往选择专门工具评估服务效果
- **测量工具承载着其创建者的价值判断**：它们不是中立的
- 一种令人不安的趋势已经形成：研究者、临床工作者和辅助专业人员使用优先减少诊断特质的测量工具——这些特质既不有害也不造成个人痛苦

## 扮装为何有害

### 什么是扮装

扮装（passing）指试图通过假装自己是未被污名化的多数群体成员来隐藏被污名化的身份。在孤独谱系语境中，扮装表现为压制孤独谱系特质以显得"与同伴不可区分"——这是 ABA 等方法的核心目标。

### 伤害证据

- 在残障和非残障语境中，扮装均与**显著伤害**相关
- 扮装努力已被识别为孤独谱系成年人**自杀风险标志**
- [[掩饰与伪装|掩饰]]（camouflaging）与多种心理健康挑战相关（焦虑、抑郁、耗竭）
- **孤独谱系扮装项目**（Autistic Passing Project）的参与者证词：
  - "我现在几乎不出门了，因为我没有精力去扮装"
  - "这些天我是为他们假装正常，不是为自己，这才是伤害"

### 举证责任反转

尽管存在这些严重伦理问题，孤独谱系结果测量仍将诊断特质的减少作为优先目标。Ne'eman 主张**举证责任应反转**：

> 因为扮装要求会施加伤害，临床工作者和辅助专业人员应**默认避免扮装**，仅在同时满足两个条件时例外：(1) 为防止伤害所必需；(2) 对孤独谱系者而言是防止伤害的**最轻负担路径**。

## 内在伤害标准

### 核心区分

当一个行为具有**内在伤害性**（如自伤）时，寻求解决它是适当的。但许多被干预瞄准的孤独谱系特质并不符合内在伤害的高标准：

| 常被干预瞄准的特质      | 内在伤害？ |
| -------------- | ---------------- |
| 缺乏眼神接触         | 无——除社会污名外通常无问题 |
| 异常韵律           | 无 |
| 手部拍打、摇晃等自我调节行为 | 无——通常用于愉悦或情绪自我调节 |
| 画特殊兴趣对象、排列物品   | 无 |
| 反复观看同一媒体       | 无 |

### "反映潜在困难"论证及其局限

有人可能主张这些特质虽非有害，但反映了有害的潜在困难——例如缺乏眼神接触与社会沟通困难相关。Ne'eman 的回应：

- 改善这些困难是合乎伦理的，但**强制典型外观很少是最有效或最轻负担的方式**
- 孤独谱系者常使用其他方式表明注意力和互惠性（如身体朝向、语言回应）
- 如果孤独谱系者从面部表情中获得的信息本就较少，眼神接触可能不会改善接收性社交沟通
- 鉴于眼神接触对孤独谱系者来说很困难，它甚至可能**分散建立关系的注意力**
- "应试教育式"地教授典型外观——在要求孤独谱系者模仿典型社交沟通的**形式**时，可能施加了一种干扰其**功能**的认知负担

### 家庭压力

家人可能出于期待孩子"看起来正常"而要求行为矫正。Ne'eman 的立场：

- 鉴于扮装会带来伤害，仅因家长希望子女看起来正常而压制孤独谱系特质是不伦理的
- 专业人员在评估行为矫正的伦理性时应考虑对他人的伤害，但必须**核实这种伤害确实存在**
- 孤独谱系者可能在某些有限情况下**选择**进行情境性扮装（如避免偏见），但这应保持为**个人选择**，而非反映在结果测量中的规范性期望

## 回应神经多样性的批评者

Manuel Casanova 等批评者暗示神经多样性反对一切干预——"有头撞、挖眼或自咬并非福气"。Ne'eman 澄清：

- 这种观点**误解了神经多样性的主张**：ND 支持者普遍支持提升孤独谱系者的适应性技能
- **神经多样性最好被理解为一种用于评估干预目标的透镜**
  - 该干预是否仅仅因其是"孤独谱系的"而试图修正某种特质？
  - 还是更适度地只针对内在有害的部分？
- 这种方法兼容于解决自伤或促进沟通
- 正如塑造唐氏综合征患者的眼睛以使其看起来正常现在被认为是野蛮的，神经多样性要求我们认识到以仅避免污名为目的强制执行典型外观的测量工具同样是不伦理的
- **压制孤独谱系特质以促进典型外观是有问题的——不论损伤程度如何**；减少个人痛苦或有害行为、或发展技能则不然

## 对孤独谱系结果测量的具体批判

### CYBOCS-ASD 案例

儿童耶鲁-布朗强迫量表-孤独谱系障碍版（CYBOCS-ASD）是一个关键案例——它已被用作临床试验的主要结果测量：

- 孤独谱系的重复行为与 OCD 的重复行为在**体验**上有根本区别：孤独谱系者通常出于愉悦或情绪自我调节而从事重复行为，而 OCD 的重复行为是痛苦的来源
- 抑制令人愉悦或有帮助的行为的伦理性显然不同于治疗令人痛苦的行为
- CYBOCS-ASD 瞄准的行为包括：手部拍打、画特殊兴趣对象、排列物品、反复观看同一媒体——这些确实是孤独谱系特质，但**并非有害特质**

### 其他测量工具

类似问题存在于其他常用孤独谱系测量中——它们将眼神接触和消除"异常"手部/手指/身体习惯与更合法的优先事项并列。

### 改革建议

- CYBOCS-ASD 等工具可**重新设计**为仅包含测量内在有害行为的项目
- 或可**仅用于以价值中立的方式探索孤独谱系**——附带明确说明：不得用于评估临床进展
- 但以当前形式使用这些工具**违反不伤害原则**——向孤独谱系者强加了不必要且可能有害的扮装要求

## 政策建议

- 需要开发新测量工具或修订现有工具以回应上述伦理关切
- 大多数现有工作关注的是主题性和心理测量学属性而**非伦理性**
- **孤独谱系者自身应参与**：CBPR（社区参与式研究）已在患者报告结果测量领域开展，类似工作需评估临床结果测量是否瞄准内在有害行为
- DSM-5 诊断标准修订中利益相关者的参与已帮助解决类似伦理问题——移除了隐性鼓励临床工作者压制孤独谱系者珍视的特殊兴趣的语言

## 更广泛的适用性

虽然这一批判在孤独谱系领域最为成熟，但与许多其他诊断也相关：当诊断特质的压制被视为适当结果时，残障者受到严重伤害。压制非典型行为可能不会带来生活质量提升——在某些情况下甚至可能**积极降低生活质量**。
