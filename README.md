## 参与贡献

无需安装任何软件，全程在浏览器完成。详见 [CONTRIBUTING.md](CONTRIBUTING.md)。

## 繁体中文

欢迎台湾、香港、澳门及海外华语使用者贡献繁体内容：

- 可新建繁体 `.md` 页面或提交繁体译本 PR
- 繁体内容不只是字形转换——术语体系、文化参照、条目表述均需独立审视
- 台湾的 CAT-Q 研究（Chun-Hao Liu 等 2026）已显示华语地区常模独立验证的必要性
