# 术语翻译对照表

本表列出神经多样性及相关领域的中文术语对应关系。**本wiki译名**为全站统一使用的术语；**其他常见中文译名**和**同义英文术语**仅供检索和对照参考。

### 考虑要点
1. 区分神经多样性事实、范式和运动三个层面，详见[[神经多样性]]。
2. 区分神经殊异、神经多元与神经多样性。神经殊异描述个体特征，神经多元描述群体异质性，神经多样性则是总体事实。
3. 区分4个D：差异（Difference）、残障（Disability）、疾病（Disease）和病症（Disorder），这在讨论ASD与去病理化时非常重要。

### 考虑要点
1. “自闭”带来”主动选择自我封闭”的误解。
2. 拒绝默认病理化的考虑。
3. 描述人时，使用”者“”个体“”群体“，如”神经殊异者“”ADHD个体“”孤独谱系障碍群体“，避免”患者“。
4. 耗竭比倦怠更能传达程度与时间尺度。

## 实践与体验

| 英文                     | 本wiki译名     | 简要定义                                                   | 其他译名           | 同义英文                                  |
| ---------------------- | ----------- | ------------------------------------------------------ | -------------- | ------------------------------------- |
| Empathy                | 同理心         | 至少包含认知和情感两个维度的心理过程                                     | 共情、同情心         |                                       |
| Double Empathy Problem | [[双向同理心问题]] | Milton (2012) 提出的理论：跨神经类型沟通断裂不是单方面"缺陷"，而是不同认知风格之间的双向错配 | 双重共情问题、双共情问题   | DEP                                   |
| Info-dumping           | [[信息大放送]]   | 孤独谱系社群中围绕专注兴趣分享大量详细信息的沟通实践——在亲和空间中被视为知识贡献而非"话题固着"      | 信息倾泻、信息倾卸、信息倾倒 | Infodumping                           |
| Camouflage/Masking     | [[掩饰与伪装]]   | 为融入神经典型社会环境而隐藏自身神经殊异特征的适应性行为——六类策略、三个维度                | 伪装行为、社交伪装      | Masking, Camouflaging                 |
| Stimming               | 调节行为        | 重复性身体动作或声音——在孤独谱系社群中被理解为情绪调节和自我安抚的自然机制，而非需要消除的“问题行为”   | 刻板行为、刺激行为、自我刺激 | Self-stimulatory behavior             |
| Self-Advocacy          | 自我倡权        | 神经殊异者以第一人称身份为自身权利和需求发声的实践——神经多样性运动的核心原则之一              | 自我倡导、自倡导       | Self-Advocate, Autistic Self-Advocacy |
### 考虑要点
1. 同理心至少包含认知和情感两个维度，而“共情”往往被理解为情感同理心维度，“同理心”易被理解成多维度。
2. DEP强调困难是两方面的，“双向”比“双重”更直观。
3. Info-dumping是积极的，大放送的色彩更接近。
4. Camouflage是正式和总括性概念，Masking是社群流行语、研究中被视为Camouflage的一个维度；因此，用“掩饰”对应Camouflage，“伪装”对应Masking。
5. Stimming的功能是自我调节。
6. Self-Advocacy常被译为倡导，但这里面有一个重要部分是争取合理权利，不仅仅是发声。

| Crip Theory                     | [[批判残障理论#酷残理论\|酷残理论]]              | 酷儿理论与残障理论在**批判性交汇**                                                           | —              | —                 |
### 考虑要点
1. 模型作为理解事物的架构，比模式更贴切。
2. Ableism的核心是以“健全”为中心，能力主义等词不容易传达出这个意味。
3. 区分损伤与残障。
